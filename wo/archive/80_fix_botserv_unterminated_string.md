# Task 80: Fix unterminated string literal in botserv_service.py

## Goal
Fix a bug in the `help` method of the `botserv_service.py` module.

## Error
When a user runs `ai do help botserv`, the server returns the following error:
`Error executing service command: unterminated string literal (detected at line 45) (botserv_service.py, line 45)`

This indicates a syntax error, likely a missing closing quote on a multi-line string, in the help text for the botserv service.

## Deliverables
- A patch for `services/botserv_service.py` that corrects the unterminated string literal.
- Verification that the `ai do help botserv` command runs without error and returns the correct help text.
