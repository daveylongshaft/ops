# Research Orangutan and Create PDF Report

## Task

Research a living orangutan and create a professional PDF report suitable for a student assignment (Physical Anthropology course).

## Requirements

### Research Content (3 pages, 8th grade level)
- Common and scientific names
- Geographic location (with map)
- Habitat description
- Population numbers
- Physical appearance
- Diet and feeding behavior
- Predators and dangers
- Social and reproductive behavior
- Endangered status
- Your personal reflection/response

### Format
- 2-3 pages of text (double-spaced preferred)
- At least 2 photos or videos of orangutans
- Map showing geographic range
- Full source citations with links
- PDF format (not Word, not Pages)

### Organization Suggestion
1. Basic description (appearance, habitat, range)
2. Behavior, diet, predators
3. Endangered status and reflection

## Tools Available
- Web search for research
- Image/map sources with proper attribution
- PDF generation tools
- Citation formatting

## Deliverable
- Final PDF file: `orangutan_research_report.pdf`
- Must include all required sections
- Properly cited sources
- Professional formatting

## Success Criteria
- At least 2 pages of content
- All required sections included (name, location, habitat, diet, behavior, status, reflection)
- 2+ photos/videos with attribution
- Map with proper labeling
- Working citations/links
- Clean PDF formatting
- 8th grade reading level
